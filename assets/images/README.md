# Image Assets — CustomerApp

Place the following image files in this folder:

| Filename         | Description                                           |
| ---------------- | ----------------------------------------------------- |
| `cm.png`         | Chief Minister photo (N. Chandrababu Naidu)           |
| `deputy_cm.png`  | Deputy Chief Minister photo (Pawan Kalyan)            |
| `corporator.png` | Ward Corporator photo (Ramakrishna — 47th Corporator) |
| `tdp_logo.png`   | TDP party logo (optional)                             |

- CM & Deputy CM shown in the Government Leaders banner on the Home screen.
- Corporator shown in the "Your Ward Representative" card.
  Recommended size: 200x200 px portrait, PNG with transparent background preferred.
